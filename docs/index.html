Features

Detects SQL Injection

(FString, Concatenation, .Format)

Detects Command Line Injections

(Shell injection and System Command Injection)

Detects Remote Code Execution

(Dynamic Code Execution Injection and Eval Injection)

Supports File Input

We currently support python and text files.

Provides Educational Feedback

SQentineL is designed to support good programming habits

by mitigating vulnerabilities in people's code and suggesting safer alternatives.

FAQs

To operate the tool in the Windows CMD, first navigate to the project root on your device. Once there, type in python -m Analyzer.Diagnostic.analyzer_test file [desiredFilePath]. This will run the chosen file through the scanner and print the vulnerabilities in the terminal.

In the event that the website backend connection is down, please do the following steps to restore functionality.

  • Install the newest version of FastAPI if neccesary.
  • Go the windows CMD and navigate to the root directory.
  • Run the command fastapi dev main.py.

SQentineL is not infallible, please use discretion when interepting its results.

About

SQentineL is a free, open-source project made by a team of three developers, Ian Bailey, Erinn Keohane, and Ryan Hayden. We are all students at West Virginia University Institute of Technology, located in Beckley, WV. Ian and Erinn are seniors in computer science and Ryan is a junior in computer engineering. The idea was created by Ian, who has an interest and knowledge in the field of cybersecurity. Our mission is to help beginner programmers develop more security conscientious applications as well as require familiarity with techniques necessary in the field.
